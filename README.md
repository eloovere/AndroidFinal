# AndroidFinal - Alarm

Configuration/Installation instructions 
The methods to use this software are very easy. Simply download the zip file to your computer. Unzip the file and open it with android studio. Run it with your choice of emulator or physical device.

Operating instructions 
Once the application is open on your device or emulator of choice operations is very straightforward. All that is required is to select a time on the spinner and hit set alarm. Afterwards the alarm will go off at the selected time.
